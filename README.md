# SineFil
An Android application with MVVM using [TMDB API](https://www.themoviedb.org/documentation/api/)

## Building
The key and TMDB API url must be added to local.properties

> **TMDB_API_KEY=** 1..1
> **TMDB_API_URL=** https://api.themoviedb.org/3/
